# Congrats Event Management - Luxury Website

Premium luxury event planning website for Congrats Event Management, Hyderabad, India.

## 🌟 Features

- **Elegant Luxury Design** - Minimalist aesthetic with warm neutral palette
- **Mobile-First Responsive** - Perfect on all devices
- **Portfolio Gallery** - Filterable image grid with lightbox
- **Client Testimonials** - Auto-rotating carousel
- **WhatsApp Integration** - Direct messaging for inquiries
- **Contact Forms** - Quick inquiry submission
- **Smooth Animations** - Professional micro-interactions
- **24/7 Availability** - Always ready for bookings

## 🛠️ Tech Stack

- **React 17+** - UI library
- **Custom CSS** - Luxury design system
- **Lucide React** - Icons
- **JavaScript ES6+** - Logic

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/mdquader69-ux/congrats-event-website.git
cd congrats-event-website
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in browser

## 🚀 Build for Production

```bash
npm run build
```
